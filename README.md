## Run Locally


Clone project
```bash
git clone git@github.com:Projet
```

Go to folder directory
```bash
cd Projet
```


Install dependencies
```bash
# yarn
yarn

# npm
npm install
```

Start server
```bash
# yarn
yarn start

# npm
npm start

# Projet run on http://localhost:3000/
```
